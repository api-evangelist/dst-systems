# DST Systems (dst-systems)

DST Systems was a provider of specialized technology, strategic advisory, and business operations outsourcing to the financial services and healthcare industries. DST was acquired by SS&C Technologies in 2018 and its products and services have since been folded into the SS&C portfolio. This repository is preserved for historical reference; current developer resources can be found through SS&C Technologies.
